Logic — The Sleeping Argument

Evaluate the following argument:


Premise 1: IF you are sleeping, THEN you are not doing homework.

Premise 2: IF you are not doing homework, THEN it is not due tomorrow.

Conclusion: Therefore, IF you are sleeping, THEN homework is not due tomorrow.


Is this argument VALID? (Does the conclusion follow logically from the premises?)

Is this argument TRUE? (Are the premises actually correct in real life?)


Important: An argument can be perfectly valid and still be completely false.

Valid logic + wrong premises = wrong answer.

This is one of the most important things they don't teach enough.


Also: Was the homework due before you did it?
